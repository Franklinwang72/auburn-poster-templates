## 用法

点开一款主题,点 **Download ZIP** 下载,上传到 **Overleaf**,设置 **Menu → Compiler → LuaLaTeX**,然后 **Recompile**。

## 你只需要编辑

- `poster-….tex` — 你的海报:先填 `\PosterSetup{title = …, authors = …}`,再写内容。`\section{…}`、`theorem`、`definition`、`lemma`、`proof`…… 可从论文原样粘贴。
- `figures/` — 你的图。

引擎部分不用动:`postertheme.sty`(全部外观)、`latexmkrc`、`fonts/`。
